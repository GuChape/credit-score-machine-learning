# Previsão de Score de Crédito com Machine Learning

## Descrição
Este projeto utiliza machine learning para classificar o score de crédito de clientes com base em dados financeiros e comportamentais.

## Funcionalidades
- Tratamento de dados
- Codificação de variáveis categóricas
- Treinamento de modelo de machine learning
- Avaliação de desempenho
- Previsão para novos clientes

## Tecnologias utilizadas
- Python
- Pandas
- Scikit-learn

## Modelo utilizado
O modelo utilizado foi o Random Forest, um algoritmo de aprendizado de máquina baseado em árvores de decisão.

## Como executar

1. Instale as dependências:
pip install -r requirements.txt


2. Execute o projeto:
python main.py


## Resultado
O modelo foi avaliado utilizando a métrica de precisão para medir seu desempenho.

## Estrutura do projeto
- `main.py`: script principal
- `clientes.csv`: base de treino
- `novos_clientes.csv`: dados para previsão

## Objetivo
Estudar como técnicas de machine learning podem ser utilizadas para prever o risco de crédito de clientes.

Ex: 82% de acerto
